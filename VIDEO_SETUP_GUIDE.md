## üåê Video Hosting Options

### Option 1: YouTube (Recommended)

**Pros:**
- Free hosting
- Unlimited bandwidth
- Mobile-friendly player
- Analytics included
- Easy embedding
- Automatic transcoding
- Subtitle support

**Cons:**
- Public by default (use "Unlisted")
- YouTube branding
- Requires Google account

**Setup Steps:**
1. Create Qbees YouTube channel
2. Upload video (set to "Unlisted")
3. Add chapters using timestamps in description
4. Enable captions/subtitles
5. Copy embed code
6. Paste into `resources/training-video.html`

**Embed Code Example:**
```html
<iframe
    src="https://www.youtube.com/embed/YOUR_VIDEO_ID?start=0&rel=0&modestbranding=1"
    frameborder="0"
    allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
    allowfullscreen>
</iframe>
```

### Option 2: Vimeo

**Pros:**
- Professional appearance
- No ads
- Better privacy controls
- Password protection
- Custom player colors
- Download protection

**Cons:**
- Free tier has limits (500MB/week upload)
- Paid plans required for more ($7-$75/month)
- Slower encoding than YouTube

**Setup Steps:**
1. Create Vimeo account
2. Upload video
3. Set privacy to "Hide from Vimeo"
4. Customize player colors (Qbees branding)
5. Copy embed code
6. Paste into `resources/training-video.html`

**Embed Code Example:**
```html
<iframe
    src="https://player.vimeo.com/video/YOUR_VIDEO_ID?title=0&byline=0&portrait=0"
    frameborder="0"
    allow="autoplay; fullscreen; picture-in-picture"
    allowfullscreen>
</iframe>
```

### Option 3: Self-Hosted

**Pros:**
- Complete control
- No third-party branding
- Works offline (if downloaded)
- No external dependencies

**Cons:**
- Server bandwidth costs
- No automatic quality switching
- Slower loading on slow connections
- Need CDN for global audience

**Setup Steps:**
1. Compress video to <2GB
2. Create folder: `assets/videos/`
3. Upload video file: `qbees-training.mp4`
4. Update `resources/training-video.html`

**Embed Code Example:**
```html
<video controls preload="metadata">
    <source src="../assets/videos/qbees-training.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>
```

**Recommended:** Use a CDN like Cloudflare or AWS CloudFront to distribute the video globally.

## üì± Mobile Optimization

### Video Encoding for Mobile

Use these settings for best mobile compatibility:

```bash
# FFmpeg command for mobile-optimized encoding
ffmpeg -i input.mp4 -c:v libx264 -preset slow -crf 22 -c:a aac -b:a 128k -movflags +faststart -vf "scale=1920:1080" output.mp4
```

**Settings Explained:**
- `-c:v libx264`: H.264 codec (universal compatibility)
- `-preset slow`: Better compression
- `-crf 22`: Good quality (18-28 range, lower = better)
- `-c:a aac`: AAC audio codec
- `-b:a 128k`: Audio bitrate
- `-movflags +faststart`: Enables progressive streaming
- `-vf "scale=1920:1080"`: Force 1080p resolution
